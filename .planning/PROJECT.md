# Wine Inventory App

## What This Is

A wine inventory application that allows users to track their personal or professional wine collection. Users can add, manage, and browse wines in their cellar, keeping records of bottle counts, vintages, producers, and tasting notes.

## Core Value

A user can quickly log a wine and know exactly what they have in their cellar at any time.

## Requirements

### Validated

(None yet — ship to validate)

### Active

- [ ] User can add a wine to their inventory with key details (name, producer, vintage, varietal, region)
- [ ] User can view their full wine inventory as a browsable list
- [ ] User can search and filter wines by varietal, region, vintage, or producer
- [ ] User can track bottle count per wine and update as bottles are consumed
- [ ] User can add tasting notes and ratings to wines
- [ ] User can mark wines as consumed or removed from inventory
- [ ] User can view wine detail page with all stored information

### Out of Scope

- Social/sharing features — v1 is personal inventory, not community
- Wine purchasing/marketplace integration — complexity, not core value
- Barcode scanning — mobile-specific, add in later version
- Advanced cellar temperature/storage tracking — hardware dependency, defer

## Context

- Greenfield project — no existing codebase
- Personal or small-business use case (home collector to small restaurant/shop)
- Wine collections can range from a few bottles to thousands
- Users want quick data entry and fast lookup
